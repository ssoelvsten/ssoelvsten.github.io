Projects

Adiar

github.com/ssoelvsten/adiar

Following up on the work of Lars Arge in 1996, Adiar is a Binary Decision Diagrams library that make use of Time-Forward Processing to achieve I/O-efficient manipulation of BDDs. This allows it to compute on BDDs that far outgrow the memory limit of the given machine.

Wordrow

wordrow.io

Wordrow is a reimplementation of the classic internet puzzle game Text Twist as a static React (PWA) webpage with a modern and sleek design. To make it static, all games are pre-computed from the entire dictionary with an implementation of the Anatree data structure in C++.
